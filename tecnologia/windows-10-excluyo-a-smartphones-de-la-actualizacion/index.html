Windows 10 excluyó a smartphones de la actualización

IMAGEN | MICROSOFT | WINDOWS |
martes 28 de julio, 2015 | 07:08 pm
           
LA NACION RECOMIENDA

Imagen de referencia.

Microsoft lanza oficialmente este miércoles el Windows 10 en 190 países, sin embargo, la actualización solo estará disponible por el momento para las PC, laptops y algunas tablets, pero no para los smartphones.

“La versión que liberamos el 29 (de julio) efectivamente es para los clientes de PC, tablet, y laptop y más adelante liberamos la versión para celulares. Todavía hay fecha por confirmar, pero sí va a ser unas semanas más adelante, no hemos confirmado la fecha”, dijo en entrevista Carlos Pérez Valdes, director de Windows Microsoft México.

CNN Expansión explica que la variedad de la fecha se debe a que la empresa priorizó la versión del sistema operativo para PC, pero sin dejar de trabajar en una versión para smartphones, de acuerdo con el directivo.

“Estamos trabajando con partners de Windows Phone 8.1 para que la actualización de Windows 10 esté disponible para la mayoría de los teléfonos a finales de este año (…) “La disponibilidad de la actualización a Windows 10 para dispositivos Windows Phone 8.1 puede variar según la operadora móvil y el operador”, indica la página web del sistema operativo.

Si tienes PC, laptop o tablet estos con los pasos a seguir antes de descargar Windows 10:

1. Respalda tus aplicaciones y archivos: Las aplicaciones, los archivos y las configuraciones migran de forma automática cuando actualizas Windows, sin embargo puede ser que algunas se pierdan, por lo que es recomendable respaldarlas ya sea en servicios en la nube como One Drive, Google Drive, Dropbox o cualquiera de tu elección.

En el caso de los antivirus y antimalware, Windows desinstalará la aplicación, pero conservará las configuraciones. Una vez completada la actualización a Windows 10, se instalará la última versión disponible de ese antivirus.

2. Sistemas actualizados: Windows 7 SP1 y Windows 8.1 Update, sino sabes si tu computadora tiene alguno de estos compruébalo a través de esta página.

Si necesitas descargar estas versiones puedes hacerlo de forma gratuita aquí para Windows 7 SP1 y aquí para Windows 8.1 Update.

3. Revisa el estado de computadora: Además de actualizar a Windows a 7 SP1 o 8.1 Update, estos son los otros requisitos mínimos que debe tener tu computadora o tablet para que Windows 10 funcione de manera óptima, según Pérez Valdes.

Procesador: De 1 GHz o más rápido o SoC

Memoria RAM de 1 gigabyte (GB) para 32 bits o 2 GB para 64 bits

Espacio en disco duro de 16 GB para un sistema operativo de 32 bits o 20 GB para un sistema operativo de 64 bits.

Tarjeta gráfica: DirectX 9 o posterior con un controlador WDDM 1.0

Pantalla: 1024 x 600

4. Haz una reservación: Una vez que tu computadora o tablet ya tenga Windows 7 SP1 o Windows 8.1 Update, verás un logo de Windows en la esquina inferior derecha de la barra de tareas, pasa el cursor o pulsa sobre él, y aparecerá un mensaje que dice: “Get Windows 10 (Consigue Windows 10)”.

Después haz clic en “Reserva tu actualización gratuita” en la ventana de la aplicación, escribe tu dirección de correo electrónico y recibirás una confirmación de la reservación.

  1. Mantente al pendiente de la notificación: Este miércoles recibirás una notificación directamente en tu dispositivo que te permitirá descargar Windows 10, puedes hacerlo otro día aunque tienes máximo un año.

  2. Ten paciencia: Carlos Pérez Valdes indicó que hay un orden en que los usuarios de Windows irán recibiendo la actualización gratuita a Windows 10.

“Vamos de alguna manera paulatinamente porque nos interesa garantizar la satisfacción del cliente. Primero serán los del grupo Windows Insider, un grupo de personas que se registraron y bajaron la versión de prueba de Windows 10 desde octubre pasado, después siguen todos los que hicieron el proceso se reserva y tienen la configuración de hardware más robusta (sus dispositivos tienen más espacio en disco duro, RAM y un mejor procesador del mínimo y después los que tienen una configuración más ligera (que csólo umple con los requisitos mínimos)”, indicó el directivo de Microsoft.

7. Espera luz verde: Una vez que descargues la versión de Windows 10, el sistema hará una revisión de tu dispositivo para comprobar que cumple con los requisitos necesarios. Podrás ver un semáforo que indica lo siguiente:

Rojo: Tu sistema no cumple con los requisitos mínimos o ya es obsoleto para la instalación de Windows 10.

Amarillo. Necesitas instalar drivers o limpiar tu disco duro o memoria Ram para tener más espacio

Verde: Tu dispositivo es óptimo para que se instale el nuevo sistema operativo.

En cada caso Windows te dirá qué hacer y te llevará de la mano para la instalación del sistema.
